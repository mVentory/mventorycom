Licensing

mVentory products are licensed under a mix of Creative Commons, Open Source and Commercial licenses. Check the licensing notes in the product to see which license applies.

Free Open Source

Some of our Magento extensions, utilies and useful code snippets are created for the benefit of the e-commerce community under one of GPL licenses. You are free to use those products for any purpose, build upon and share in the original form or with your modifications.

Most of our open source code can be found in our GitHub Repositories.

Limited Open Source

Some of our products have the source code published, but their use is not free for commercial purposes. They are released under Creative Commons BY-NC-ND license. It allows you to inspect the code and try the product, but you cannot use it for commercial purposes.

Limitations
  • Sharing: you can share the code in the original unchanged form if you link back to our source repositories.
  • No derivatives: you can make changes to the code, but you cannot share them with others. We will appreciate if you make a pull request back to our repository.
  • Non-commercial: you cannot use the product if your activities fall under this definition:

    A commercial use is one primarily intended for commercial advantage or monetary compensation.

Commercial License

The license allows you to use the licensed product within your organisation in perpetuity with one limitation – you cannot share the product or its source code in any shape or form outside of your organisation, including any modifications you made.

Modifications of the code do not have to be contributed back to us, but it may be a good idea to include any improvements in the main branch so that you get them with new updates.

The full text of mVentory Commercial License is published on GitHub. You can track any changes in the text back to the time when the license was granted in the repository history.
