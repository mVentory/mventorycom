Installation

mVentory system consists of an Android application and a Magento extension that allows the phone to access Magento through the XML-RPC API.

Magento extension installation

Magento extension installation process is extremely simple. Once you have obtained the extension key from Magento Connect, follow the standard steps of Magento extension installation process. Once it is installed, you are ready for configuration.

  1. Log into Admin panel
  2. Go to System -> Magento Connect -> Magento Connect Manager menu.
  3. Log in with admin account if prompted.
  4. Under Install New Extensions, paste the extension key and click Install button.
  5. View the installation output in the console and check that Package has been installed.
  6. Press Refresh button (bottom left).
  7. Check that the extension (MVentory) now appears in the list (at the bottom).
  8. Click on Return to Admin (top right)
  9. Click on Log out (top right) and log in again (to flush and reload permissions).

Android Application Installation

Android application can be installed through Google Play Store.
